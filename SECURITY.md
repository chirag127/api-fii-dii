# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | Yes       |

## Scope

This repo scrapes publicly available NSE/MoneyControl data and serves static JSON. It handles no user accounts, no secrets, and no personal data. The scrape script reads external URLs and writes JSON files; no user input is processed.

## Reporting a Vulnerability

Use [GitHub Security Advisories](https://github.com/chirag127/fii-dii-activity-api/security/advisories/new) to report privately. Include:

- Description of the issue
- Steps to reproduce
- Potential impact

Expect acknowledgement within 7 days. Public disclosure after a fix is merged.
